Welcome to CVTron
=======================================


.. image:: brief/logo.png
  :scale: 30 %
  :align: center
  :target: https://github.com/cv-group/cvtron

`CVTron`_ is a Computer Visoin library Based on `Google TensorFlow <https://www.tensorflow.org>`_ and `TensorLayer <https://www.github.com/tensorlayer/tensorlayer>`_.  It provides state-of-the-art models and algorithms for building industry level computer vision applications.
More details can be found `here <https://github.com/cv-group/cvtron>`_.
